Innoventory Drive - PHP version (created by assistant)
How to use:
1. Put the 'innoventory-drive' folder into C:\xampp\htdocs\
2. Start Apache and MySQL from XAMPP Control Panel.
3. Open http://localhost/phpmyadmin and run the SQL file (innoventory_sql.sql) to create the database and users table.
4. Go to http://localhost/innoventory-drive/login.php
5. Create an admin manually in the database to approve users, or insert an admin row.

Files included:
- index.php, login.php, signup.php, pending.php, approval.php, denied.php, login-error.php, status.php
- /php: connect.php, login-action.php, signup-action.php, approve-user.php, logout.php
- /css/style.css, script.js
